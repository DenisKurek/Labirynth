Detailed Description

Class designed to load and set up shaders.

Constructor & Destructor Documentation

◆ Shader()

Shader::Shader ( const char *  vertexFilePath,
const char *  fragmentFilePath 
)

The constructor that initializes a new Shader object and makes it ready to be used.

This process involves a few stages. The shader source codes are read from the files, then they are compiled, linked together, attached, and a complete shader program is created as a result.

Parameters
vertexFilePathA path to the file that stores the vertex shader source code.
fragmentFilePathA path to the file that stores the fragment shader source code.
Note
On failure, prints more details to the standard error output.
